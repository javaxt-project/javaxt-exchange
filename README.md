# JavaXT Exchange
The javaxt-exchange library is used to retrieve, create, edit, and delete Email Messages, Contacts, and Calendar Events hosted on a Microsoft Exchange Server. 

## Project homepage
Information about the javaxt-exchange library, documentation, and releases can be found on the project homepage:
 - <a href="https://www.javaxt.com/javaxt-exchange/">https://www.javaxt.com/javaxt-exchange/</a>
